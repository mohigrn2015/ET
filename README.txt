Note: This project, I have use generic repository pattern.

First you have clone the project from git.
1. First, change Data Source name from appsettings.json,
2. update database by command "update-database" from package manager console. 
3. Run the application
4. Add category from "Expense Category" from nav menu. After add you can edit or delete.
5. Then click "Daily Expense" from the nav menu here you can add, edit and delete particular expense.
6. From nav menu click "Search" button, here you can see the expense within two selected date.
